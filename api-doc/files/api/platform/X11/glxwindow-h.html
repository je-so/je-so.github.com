int init_glxwindow(

   const struct x11window_it *eventhandler,
   uint8_t nrofattributes,

)

Initializes glxwin and creates a hidden on window on x11screen.  The configuration is read from the array configuration[nrofattributes].  To handle any events set eventhandler to a valid value != 0.  Call show_glxwindow to make the window visible to the user.
