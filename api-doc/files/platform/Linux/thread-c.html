thread_startargument_t

struct thread_startargument_t

Startargument of the started system thread.  The start function <thread_t.startpoint_thread> then calls the main task of the thread which is stored in thread_t.task.

nr_threads

uint32_t nr_threads

Contains the number of threads in this group.  All threads share the same thread_main function and the same argumen at the beginning.

isAbort

volatile bool isAbort

Indicates if not all threads could have been started successfully.  Ensures transactional behaviour.

isFreeEvents

bool isFreeEvents

Indicates to a thread that it is responsible to free all event semaphores.

isfreeable_semaphore

semaphore_t isfreeable_semaphore

Threads signal on this event before entering the main function.  The first thread waits on this event to know for all other threads have entered the main function.  Therefore it is safe to free the two event semaphores which are no longer of use.

isvalid_abortflag

semaphore_t isvalid_abortflag

Threads wait on startup for this event.  After this event has occurred variable isAbort contains the correct value.

Functions

signalstacksize_threadstack

static inline unsigned signalstacksize_threadstack(void)

Returns the minimum size of the signal stack.  The signal stack is used in case of a signal or exceptions which throw a signal.  If for example the the thread stack overflows SIGSEGV signal is thrown.  To handle this case the system must have an extra signal stack cause signal handling needs stack space.

threadstacksize_threadstack

static inline unsigned threadstacksize_threadstack(void)

Returns the minimum size of the thread stack.  The stack should be protected against overflow with help of protected virtual memory pages.

framestacksize_threadstack

static inline size_t framestacksize_threadstack(void)

Returns the size of a stack frame for one thread.  The last tail protection page is not included.
