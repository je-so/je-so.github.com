search

findnode_redblacktree

Searches for a node with equal key.  If it exists it is returned in found_node else ESRCH is returned.

change

insertnode_redblacktree

Inserts a new node into the tree only if it is unique.  If another node exists with the same key as new_key nothing is inserted and the function returns EEXIST.  If no other node is found new_node is inserted The caller has to allocate the new node and has to transfer ownership.

removenode_redblacktree

Removes a node if it has equal key.  The removed node is not freed but a pointer to it is returned removed_node to the caller.

updatekey_redblacktree

extern int updatekey_redblacktree(
   redblacktree_t *tree,
   const void *old_key,
   const void *new_key,
   const redblacktree_update_key_t *update_key,
   const redblacktree_compare_t *compare_callback
)

If a node exists in the tree with the same key as old_key then it is removed from the tree and update_key_f is called with new_key and the pointer to the removed node as its parameter.  If update_key_f returns success the updated node is inserted.  If the removed node can not be inserted cause new_key exists already update_key_f is called a second time to revert the changes (with old_key as first parameter) and the node is re-inserted at its old place.
