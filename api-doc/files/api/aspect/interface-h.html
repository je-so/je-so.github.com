Parameter

declared_oitThe name of the structure which is declared as the interfaceable object.
object_tThe type of the object which is the first parameter of the service interface functions.  A pointer to this object is stored in the declared structure.
interface_itThe type of the service interface the stored object offers.  A pointer to this service interface is stored in the declared structure.

interface_oit

struct interface_oit

A template for a generic object and its exported interface.

The structure contains a pointer to the object and a pointer to a table of pointers to functions.

With help of the function table the exported interface of the object can be accessed.

This structure serves as an example of the structure of interfaceable objects.
