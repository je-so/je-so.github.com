Include files

Linux specific includes

Include all C99, Posix and Linux specific header files.  This ensures that all files are compiled in the same way.  No system specific includes are used in any implmentation file.

May be changed

This rule may be changed in the future to make up for faster compilation.

!OVERWRITE! system specific types

sys_filedescr_t

#define sys_filedescr_t int

Type represents a system file descriptor.  This type is Posix specific.  Type represents also a network connection.

sys_filedescr_INIT_FREEABLE

#define sys_filedescr_INIT_FREEABLE (-1)

Static initializer for a file descriptor which is not valid.

sys_mutex_t

#define sys_mutex_t pthread_mutex_t

Type represents a mutual exclusion lock.  This type is Posix specific and needs pthread support.

sys_mutex_INIT_DEFAULT

#define sys_mutex_INIT_DEFAULT PTHREAD_MUTEX_INITIALIZER

Static initializer for sys_mutex_t.

sys_process_t

#define sys_process_t pid_t

Type represents a system process.  This type is Posix specific.

sys_process_INIT_FREEABLE

#define sys_process_INIT_FREEABLE ()

Static initializer for sys_process_t.

sys_semaphore_t

#define sys_semaphore_t int

Types represents a system semaphore.  This type is Posix specific.

sys_semaphore_INIT_FREEABLE

#define sys_semaphore_INIT_FREEABLE (-1)

Static initializer for sys_semaphore_t.

sys_socketaddr_t

#define sys_socketaddr_t struct sockaddr

Type which holds a generic socket address.  This type is defined as Posix specific struct sockaddr.

sys_socketaddr_MAXSIZE

#define sys_socketaddr_MAXSIZE sizeof(struct sockaddr_in6)

Value which describes the maximum size of all versions of ip socket addresses.  This type is defined as Posix specific sizeof(struct sockaddr_in6).

sys_thread_t

#define sys_thread_t pthread_t

Types represents a system thread.  This type is Posix specific and needs pthread support.

sys_thread_INIT_FREEABLE

#define sys_thread_INIT_FREEABLE ()

Static initializer for sys_thread_t.
