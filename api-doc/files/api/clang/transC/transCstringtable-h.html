next

transCstringtable_entry_t * next

Points to next data block.  See strsize how to detect the end of a string and the beginning of a new string.

strsize

uint16_t strsize

Only 15 bit of strsize are used to describe the size.  Therefore the supported size of a single data block is always <= 32767.  Bit 15 (value 32768) is used to mark a block as an extension of the previous data block.  It belongs to the same string as the previous block if bit 15 is set.

transCstringtable_t

struct transCstringtable_t

Stores string literals used for initialization.  The stored values contain no more escape sequences.  They must have been replaced by the corresponding byte sequence of the UTF-8 encoded codepoint before inserting a string.

insertstring_transcstringtable

int insertstring_transcstringtable(transCstringtable_t *strtable,
/*out*/void **strid,
/*out*/uint8_t **addr,
uint8_t size)

Adds a new string to the table.  Only memory is reservered.  The size of the string is determined by size and the address of the reserved memory block is returned in addr.  The id of the new string is returned in strid.  You can later use strid to iterate over the string content.

appendstring_transcstringtable

int appendstring_transcstringtable(transCstringtable_t *strtable,
/*out*/uint8_t **addr,
uint8_t size)

Increases last inserted string by size bytes.  The start address of the appended memory is returned in addr.  It is possible that the string data is split over multiple memory pages and therefore non contiguous.

shrinkstring_transcstringtable

int shrinkstring_transcstringtable(transCstringtable_t *strtable,
uint8_t *endaddr)

Shrinks the last inserted or appended string memory block.  The values addr and size describe the parameter given or returned from the last call to insertstring_transcstringtable or appendstring_transcstringtable.  The endaddr must be in the range [addr, addr + size].  An endaddr of set to addr shrinks the last memory block to size 0.  An endaddr set to value (addr+size) does not change the size of last inserted or appended string data block.
